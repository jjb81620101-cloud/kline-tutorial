# 📊 K線教學軟體

一個互動式網頁應用，幫助初學者學習股市 K線圖表的判讀方法。

## 功能特色

- **📖 系統化教學**：3 堂循序漸進的教學課程
  - 第 1 課：K線基本概念
  - 第 2 課：K線基本形態
  - 第 3 課：K線與趨勢

- **🎯 互動式測驗**：10 道隨機試題，涵蓋多種難度
  - 基礎判讀
  - 形態識別
  - 趨勢判斷
  - 買賣信號

- **📈 進度追蹤**：自動保存學習進度和成績
  - 實時成功率顯示
  - 完成日期記錄
  - 成績統計

- **🎓 智能評估**：直到成功率達 80% 才算完成
  - 低於 80% 自動重新測驗
  - 實時反饋（對錯說明）
  - 進度百分比顯示

## 使用方法

### 線上使用
直接在瀏覽器中打開 `index.html` 或訪問 GitHub Pages 地址

### 本地使用
1. 克隆或下載此倉庫
2. 用瀏覽器打開 `index.html`
3. 點擊「開始教學」按鈕

## 軟體結構

```
kline-tutorial/
├── index.html      # 主頁面
├── styles.css      # 樣式設定
├── script.js       # 應用邏輯
└── README.md       # 說明文檔
```

## 課程內容

### 第 1 課：什麼是 K線？
- K線的基本概念（開盤價、最高價、最低價、收盤價）
- K線的組成（實體、上影線、下影線）
- K線的顏色（陽線和陰線）

### 第 2 課：K線的基本形態
- 陽線（上漲）
- 陰線（下跌）
- 十字線
- 錘子線（看漲信號）
- 倒錘子線（看跌信號）

### 第 3 課：K線與趨勢
- 上升趨勢特徵和買點
- 下降趨勢特徵和賣點
- 震盪趨勢特徵和交易策略

## 測驗題型

- **基礎判讀**：識別 K線的漲跌方向
- **形態識別**：認識特殊的 K線形態
- **趨勢判斷**：判斷當前的市場趨勢
- **買賣信號**：根據 K線做出交易決策

## 學習建議

1. **認真學習每一課**：基礎知識很重要
2. **理解而非死記**：理解 K線背後的市場含義
3. **反覆練習**：通過測驗鞏固知識
4. **結合實盤**：在真實股票走勢中應用

## 成功率計算

- 每答對 1 題 +1 分
- 總分 = 答對題數 / 總題數 × 100%
- **目標**：達到 80% 或以上
